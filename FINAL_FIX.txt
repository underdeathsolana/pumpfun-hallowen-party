User melihat HTML TAGS bukan rendered HTML!

ROOT CAUSE:
-----------
❌ Function typeWriter() melakukan typing CHARACTER BY CHARACTER
❌ Ini break HTML tags karena type: < lalu h lalu 4 lalu >
❌ Browser tidak bisa parse HTML yang incomplete
❌ Jadi ditampilkan sebagai plain text

CODE SEBELUMNYA:
----------------
```javascript
setTimeout(() => {
    typingIndicator.style.display = 'none';
    resultText.style.display = 'block';
    typeWriter(suggestion, resultText); // ❌ Type char by char
}, 1500);
```

SOLUSI:
-------
✅ Langsung set innerHTML tanpa typing effect
✅ Ganti dengan fade-in effect yang smooth
✅ HTML langsung di-render dengan benar

CODE SETELAH PERBAIKAN:
-----------------------
```javascript
setTimeout(() => {
    typingIndicator.style.display = 'none';
    resultText.style.display = 'block';
```
